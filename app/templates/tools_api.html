GODOT – Tools

Application programming interface (API)

The GODOT API allows for accessing data in the GODOT graph. the generic URI for querying is as follows:

https://godot.date/api/v1/query?id={GODOT ID}

The JSON response consists mainly of an English translation of the date and a list of attestations:

{
  "date_translation": "...",
  "attestations": [...]
}

Example: https://godot.date/api/v1/query?id=https://godot.date/id/WbFCQ7e2jgvCawNkp22ixQ
